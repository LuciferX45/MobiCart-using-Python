Basic Information about the files in the MobiCart Folder:
1)main.py - contains the main program, all the other files support the functions in this program
2)Cart.py - contains the code for the functions performed for the Cart
3)Details.xlsx - all the data is stored in this excel sheet
4)LinkedList.py - contains the code for the Linked List 
5)images - contains the image for the AddtoCart button
